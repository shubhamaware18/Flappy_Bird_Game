## Usage

- To play the game manually:
  -  Run `python game.py`
  -  Click or press <kbd>Space</kbd> to start the game.
  -  Press <kbd>Space</kbd>or<kbd>UP</kbd> to control the bird.
  -  Press <kbd>Escape</kbd> to exit the game.
